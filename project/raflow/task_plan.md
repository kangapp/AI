# 任务规划 - RaFlow 语音交互工具

> 基于 Tauri v2 与 ElevenLabs Scribe v2 Realtime API 的实时语音转录工具

---

## 项目概述

**目标**: 构建类似 Wispr Flow 的系统级语音听写工具，实现：
- 全局热键触发录音
- 实时语音转文字（<150ms 延迟）
- 智能文本注入到当前光标位置
- 不可输入时回退到剪贴板

**技术栈**: Tauri v2 + Rust + React + ElevenLabs Scribe v2
